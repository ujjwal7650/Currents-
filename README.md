# 🌍 Currents – Live FX & Crypto Dashboard

Currents is a modern real-time Forex and Cryptocurrency dashboard that provides live exchange rates, crypto prices, historical charts, currency conversion, portfolio tracking, watchlists, and market insights using public APIs.

## 🚀 Features

- 📈 Live Forex Exchange Rates
- ₿ Live Cryptocurrency Prices
- 🔄 Currency & Crypto Converter
- 📊 Interactive Historical Price Charts
- ⭐ Watchlist with Local Storage
- 💼 Portfolio Tracker with Live Valuation
- 🔥 Trending Cryptocurrencies
- 📉 Top Gainers & Losers
- 🚀 Momentum Ranking (Top 50 Cryptocurrencies)
- 📱 Fully Responsive UI
- ⚡ Automatic Data Refresh Every 60 Seconds
- 🎨 Modern Dark Theme Dashboard
